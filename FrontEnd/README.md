Instruccion para guardar en GIT
  1- guardar todos los cambios en los archivos abiertos,
  2- en la terminal cmd ubicarse en la raiz de la carpeta SYMATRI, no backend ni frontend
  3- comando => git add .
  4- Enter
  5- comando => git commit -m "Escriba descripcion de lo que va a guardar, ejemplo:
                                Agregar ventana dashboard profesor"
  6- Enter
  7- comando => git push origin main
  8- Enter
